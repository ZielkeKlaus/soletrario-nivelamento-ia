# Soletrário 🐹📚

O *Soletrário* é um jogo pedagógico desenvolvido para a plataforma Android, focado no auxílio ao aprendizado de gramática e ortografia da língua portuguesa para crianças do ensino fundamental.

O projeto faz parte da disciplina de Projeto de Desenvolvimento I do curso de Análise e Desenvolvimento de Sistemas da *UniSenac Pelotas*.

## 🚀 Objetivo do Projeto
Proporcionar uma ferramenta lúdica e acessível para que crianças possam praticar exercícios gramaticais de forma divertida e interativa.

## 🎨 Identidade Visual e Mascote
O mascote oficial do jogo é o *Nico*, uma capivara carismática que acompanha o aluno durante toda a jornada de aprendizado.

## 📖 Conteúdo Pedagógico
O jogo abordará 5 temas principais de exercícios:
1. *Sílaba tônica*
2. *Uso de X / CH*
3. *Uso de R / RR*
4. *Uso de SS / Ç*
5. *Acentuação*

## 🛠️ Tecnologias Planejadas
- *Framework :* ReactNative (JavaScript/TypeScript)
- *Banco de Dados:* PostgreSQL
- ⁠*Gerenciamento de Estado:* Context API / Redux
- *Design:* Figma
